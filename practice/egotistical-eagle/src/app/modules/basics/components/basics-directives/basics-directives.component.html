Directives

  • instructions in the DOM
  • attribute directives: look like normal HTML attribute, possibly with data binding or event binding; only affects element they are added to
  • structural directives: look like normal HTML attribute, but has a leading * (for desugaring): affects a whole area in DOM (elements added/removed)
  • CLI: ng g d directive-name
