Binding

String Interpolation

  • {{ allowNewPet }}   {{ allowNewPet }}
  • use if you want to output something in your template or print some text

Property Binding

  • <p [innerText]="allowNewPet"></p>
  • use if you want to change some property of an HTML element, directive, or component

Event Binding

  • (event)="myFunction($event)"
  • {{ petName }}

Two Way Data Binding

  • need to import FormsModule from @angular/forms to use [(ngModel)]
  • {{ petName2 }}

*ngIf="expression; else myLocalRefBlock"

  • <ng-template #myLocalRefBlock>
  • the # defines a local reference (think of it as a marker)
  • can be used everywhere in template but cannot be used in Typescript code, unless @ViewChild decorator is used (but generally not recommended practice to directly access DOM elements)
